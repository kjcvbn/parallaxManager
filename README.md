#ParallaxManager

시차 모션 라이브러리 입니다.


### Usage

```html
<section class="pxm">
    <div data-pxm="y:0~600/x:0~600"></div>
    <div data-pxm="y:600~0/x:600~0"></div>
</section>

```



###Option

| option | description |
| :----- | :---------- |
| x      | translateX  |
| y      | translateY  |
| z      | translateZ  |
| r      | rotate      |
| s      | scale       |
| o      | opacity     |
| etc..  | 변환이 필요없는 Css |



### Dependencies
* jQuery >=1.8.3
